--- title: Hello, World. date: 2012-08-21 13:24:28 summary: In which I greet you, gentle reader. --- With some consternation, I've overhauled my website. Fun fact: I helped put the e-book together for [A List Apart](http://alistapart.com) back when I worked for Threepress. The code for this website is hosted on [GitHub](https://github.com/samjacoby/samjacoby.com). Not much to see, but knock yourself out if you're so inclined.
